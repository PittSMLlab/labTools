extractStimArtifactIndsFromTrigger

PURPOSE

EXTRACTSTIMARTIFACTINDSFROMTRIGGER Extract TAP stim artifact peak indices.

SYNOPSIS

function indsStimArtifact = extractStimArtifactIndsFromTrigger(times, rawEMG_TAP, pinHreflexStim, options)

DESCRIPTION 

EXTRACTSTIMARTIFACTINDSFROMTRIGGER Extract TAP stim artifact peak indices.

   Extract the indices of the stimulation artifact peaks in the proximal
 tibialis anterior EMG signal (more robust than calf muscles during
 walking) using the rising edge of the stimulation trigger pulse to
 localize the artifact peak.

 Inputs:
   times          - number of samples x 1 array of time in seconds from
                    the start of the trial for each sample
   rawEMG_TAP     - 2-element cell of number of samples x 1 arrays for
                    right (cell 1) and left (cell 2) proximal TA EMG
                    (if one cell is empty, that leg is skipped)
   pinHreflexStim - 2-element cell of number of samples x 1 arrays for
                    right (cell 1) and left (cell 2) stim trigger pulses

 Optional Name-Value Inputs:
   threshStim      - stim trigger pulse detection threshold, V
                     (default: 2.5)
   winDurStim      - search window duration around trigger pulse, s
                     (default: 0.1)
   minArtifactPeak - minimum stim artifact peak height, V (default: 0.001)

 Outputs:
   indsStimArtifact - 2 x 1 cell of number of stimuli x 1 arrays for
                      right (cell 1) and left (cell 2) stim artifact
                      indices

 Toolbox Dependencies:
   None

 See also HREFLEX.EXTRACTSNIPPETS, COMPUTEHREFLEXPARAMETERS.
