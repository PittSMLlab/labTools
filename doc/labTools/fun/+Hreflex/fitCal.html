fitCal

PURPOSE

FITCAL Fit M- and H-wave recruitment curves.

SYNOPSIS

function fit = fitCal(intensitiesStim, amplitudesWaves)

DESCRIPTION 

FITCAL Fit M- and H-wave recruitment curves.

   Accept stimulation intensities and H- and M-wave amplitudes for the
 right and left legs and fit a modified hyperbolic function to the M-wave
 data (equation 1) and an asymmetric Gaussian to the H-wave data
 (modified from equation 2), based on Brinkworth et al.
 (J. Neurosci. Methods, 2007), Section 2.4 Curve Fitting.

 Inputs:
   intensitiesStim - 2 x 1 cell of number of stimuli x 1 arrays for
                     right (cell 1) and left (cell 2) leg stimulation
                     amplitudes in mA
   amplitudesWaves - 2 x 2 cell of number of stimuli x 1 arrays for
                     right (row 1) and left (row 2) leg H-reflex
                     amplitudes: M-wave (col 1), H-wave (col 2)

 Outputs:
   fit - struct of M-wave and H-wave fit function, optimized parameters,
         and maximum fit M value for normalization

 Toolbox Dependencies:
   Statistics and Machine Learning Toolbox (nlinfit)
   Signal Processing Toolbox (findpeaks)

 See also HREFLEX.PLOTCAL, GENERATEHREFLEXRECRUITMENTCURVES.

0119 % information criterion (BIC) for a fitted recruitment curve model.
0120 %
0121 % Inputs:
0122 %   intensities - number of data points x 1 array of stimulation
0123 %                 intensities (mA)
0124 %   amps        - number of data points x 1 array of wave amplitudes (mV)
0125 %   func        - function handle for the model: func(params, intensities)
0126 %   params      - 1 x numParams vector of fitted model parameters
0127 %
0128 % Outputs:
0129 %   R2  - coefficient of determination
0130 %   AIC - Akaike information criterion
0131 %   BIC - Bayesian information criterion
0132 %
0133 % Toolbox Dependencies:
0134 %   None
0135 
0136 numParams = numel(params);                      % number of parameters
0137 numPnts   = numel(amps);                        % number of data points
0138 pred      = func(params, intensities);          % model-predicted values
0139 residuals = amps - pred;
0140 SSR = sum(residuals.^2);                        % sum of squared residuals
0141 R2  = 1 - (SSR / sum((amps - mean(amps)).^2)); % R^2 calculation
0142 % Akaike information criterion

0144 % Bayesian information criterion
0145 BIC = numPnts * log(SSR / numPnts) + numParams * log(numPnts);
