DESCRIPTION 

medianFilter  Applies median filter

   this = medianFilter(this, N) applies median filter of order N

   Inputs:
       this - labTimeSeries object
       N - filter order (must be odd)

   Outputs:
       this - filtered labTimeSeries with NaN at edges
