SYNOPSIS

function [J,B,Xh,V] = estimateDyn(X, realPolesOnly, nullB, J0)

DESCRIPTION 

estimateDyn for a given vector X, it estimates matrices J,B,V such that
Xh(:,i+1)=J*Xh(:,i)+B; Xh(:,1)=1; and X~V*Xh where J is Jordan Canonical Form
